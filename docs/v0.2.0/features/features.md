# Features

Verdocs comes packed with professional-grade features designed to make documentation creation and consumption a seamless experience.

| Feature | Description |
| :--- | :--- |
| [Search Engine](./search.md) | High-speed, client-side searching across all documentation versions. |
| [Minimap (TOC)](./minimap.md) | Automatically generated "On this page" navigation for every page. |
| [Sidebar Management](./sidebar.md) | Intelligent, folder-based navigation with priority sorting. |
| [Code Blocks](./code-blocks.md) | Optimized syntax highlighting and developer UX enhancements. |
| [Preview Server](./preview-server.md) | Integrated development server with robust hot-reloading support. |
